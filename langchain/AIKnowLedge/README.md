# 应用 1：【初级】智能网页/PDF 知识库（“极速版”RAG）
目标： 掌握数据流的闭环（加载 -> 切分 -> 向量化 -> 检索 -> 回答）。

场景描述： 你给它一个长篇技术文章的 URL 或一份 PDF 简历，然后直接针对内容提问。

# 核心技术栈： * Crawl4AI (最新的网页解析)

FAISS (本地向量存储)
ChatOpenAI (模型)
LCEL (基础链)

# 学习重点：
理解为什么需要 Embedding（向量化）。
学会如何通过 Prompt 约束 AI 不要胡编乱造。
为什么选它： 这是所有 AI 应用的基础。如果你连这个都跑不通，后面的复杂应用会像空中楼阁
数据库采用 Milvus 向量数据库.
